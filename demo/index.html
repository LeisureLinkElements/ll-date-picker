An example of <ll-time-picker>

When time is set via picker...

  • the input's value is set to standard 12-hour AM/PM format; eg: "6:30 PM"
  • the data-iso attribute is set to the ISO format; eg: "18:30:00"
  • the data-iso attribute is two-way bound, so you can bind it to a property of the Polymer element that contains this input; eg:
    <input is="ll-time-input" data-iso={{property.checkInTime}}/>;
